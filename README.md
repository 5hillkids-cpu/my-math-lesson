# Math Quest

Interactive lesson website with multiple elementary math decks, built as a static React app that runs directly in the browser.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Add that repository as the `origin` remote.
3. Rename the local branch to `main`.
4. Push the code to GitHub.
5. In GitHub, open `Settings > Pages` and ensure `GitHub Actions` is the source.

Example commands:

```powershell
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO.git
git add .
git commit -m "Initial Math Quest site"
git push -u origin main
```

## Expected Link

Your public GitHub Pages link will usually be:

`https://YOUR-USERNAME.github.io/YOUR-REPO/`

If you use a user site repository named `YOUR-USERNAME.github.io`, then the link becomes:

`https://YOUR-USERNAME.github.io/`

## Local Entry

Open `index.html` locally to view the site without deployment.
